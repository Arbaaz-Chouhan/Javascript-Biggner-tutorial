# shift();

```javascript
var a= ["Arbaaz","Adnan","Fayyaz","Danis"];
document.write(a+ "<br>");
a.shift();
document.write(a+ "<br>");
```
---
# unshift();
```javascript
var a= ["Arbaaz","Adnan","Fayyaz","Danis"];
document.write(a+ "<br>");
a.unshift("Rahman");
document.write(a+ "<br>");
```
