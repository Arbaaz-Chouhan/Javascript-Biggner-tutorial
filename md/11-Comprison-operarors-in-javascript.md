## COMPARISON OPERATOR
|OPERATOR|DESCRIPION|
|--------|----------|
|==|EQUAL TO|
|!=|NOT EQUAL|
|!==|NOT EQUAL VALUE OR NOT EQUAL TYPE|
|>|GREATER THEN|
|<|LESS THEN|
|>=|GREATER THEN OR EQUAL TO|
|<=|LESS THEN OR EQUAL TO|
---
>we can compare any thing with the help of this table and find a value in (true) or (false) so the pattern is

```javascript
var a = 10;
var b = 10;
console.log(a == b);

var a = 10;
var b = "10";
console.log(a === b );

console.log(2 != 3 );
console.log(2 !=="2");

console.log(20 >= 5);
console.log(20 >= 20);

console.log(20 <= 20)
console.log(20 < 20)

```
