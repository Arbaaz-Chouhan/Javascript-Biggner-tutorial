> table of Assignment operater

|operator|example|same as|
|--------|-------|-------|
|=|x=y|x=y|
|+=|x+=y|x=x+y|
|-=|x-=y|x=x-y|
|*=|x*=y|x=x*y|
|/=|x/=y|x=x/y|
|%=|x%=y|x=x%y|
|**=|x**=y|x=x**y|
>we can use assingnent opertor as a compare and give some value so the pattern to write this in input is
```javascript
var a = 20;
var b = 10;
 a += b;
document.write(a);
document.write("<br>");

var a = 15;
var b = 10;
a -= b;
document.write(a);
document.write("<br>")

var a = 10;
var b = 10;
a *= b;
document.write(a);
document.write("<br>");

var a = 64;
var b = 8;
a /= b;
document.write(a);
document.write("<br>");

var a = 10;
var b = 3;
a %= b;
document.write(a);
document. write("<br>")

var a = 10;
var b = 3;
a **= b;

```
